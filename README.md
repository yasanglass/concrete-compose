## Download

1. Add it in your root build.gradle at the end of repositories

```groovy
allprojects {
	repositories {
		...
		maven { url 'https://jitpack.io' }
	}
}
```

2. Add the dependency

```groovy
dependencies {
    implementation 'com.github.yasanglass:stillbirth-android:<version>'
}
```
